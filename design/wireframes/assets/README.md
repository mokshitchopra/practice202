## ğŸ¨ Asset Guidelines

### Icons
- **Format**: SVG preferred, PNG for complex icons
- **Size**: 16px, 20px, 24px, 32px variants
- **Style**: Outline and solid versions
- **Library**: Heroicons or similar consistent set

### Images
- **Placeholders**: Use consistent placeholder images
- **Format**: JPEG for photos, PNG for graphics
- **Size**: Optimized for web (max 2MB)
- **Aspect Ratios**: 4:3 for items, 1:1 for profiles

### Exports
- **Format**: PNG for screenshots, PDF for documentation
- **Resolution**: 2x for retina displays
- **Naming**: Descriptive filenames with version numbers
- **Organization**: Group by screen type and version
